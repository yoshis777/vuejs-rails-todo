## 概要

[Vue.jsとRailsでTODOアプリのチュートリアルみたいなものを作ってみた](https://qiita.com/naoki85/items/51a8b0f2cbf949d08b11)を参考にTODOアプリを実装した。

* RailsでVue.js用のRestAPI作成
* Vue-RouterによるURLルーティング
* Vue.jsによるデータバインディング
* Vue.jsによるajax経由のDBデータのCRUD操作
